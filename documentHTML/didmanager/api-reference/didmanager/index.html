# DIDmanager

DID manager SDK for manage DIDs

# Main function

# createDID 

dids.createDID(claimType, data, privateKeySigner, privateKey);

Create DID contract.

# Parameters

  1. claimType - ClaimType : DID type, default is ClaimType.PATIENT (check DID type)
  2. data - string : Init data for different claim, please use user id of off-chain side or use random number as id
  3. privateKeySigner - string : Private key of signer of claimHolder, get this from server side
  4. privateKey - string : Private key of contract creator - owner of identity contract which will be created

# Returns

  1. receipt - object : A transaction receipt object, or null if no receipt was found. (check receipt)
  2. eventLogs - object : Event logs of transaction
  3. claimId - string : Added claimId of DID contract

# Example 

dids.createDID(ClaimTypes.PATIENT,
    "PATIENT VERIFIED",
    signerKey,
    account.privateKey
  );

# addClaim 

dids.addClaim(
  privateKey,
  privateKeySigner,
  targetIdentity,
  claimType,
  scheme,
  issuer,
  data,
  [uri]
);

Add claim to DID contract

# Parameters

  1. privateKey - string : Private key of account that has permission to interact with DID
  2. privateKeySigner - string : Private key of account that has permission to interact with DID
  3. targetIdentity - string : DID contract address
  4. claimType - ClaimTypes : The type of claim
  5. scheme - number : The scheme of claim
  6. issuer - string : The issuer address of claim
  7. data - string : The raw data of claim
  8. uri - string (optional): The uri of claim

# Returns

  1. receipt - object : A transaction receipt object, or null if no receipt was found. (check receipt)
  2. eventLogs - object : Event logs of transaction
  3. claimId - string : Added claimId of DID contract

# Example 

dids.addClaim(
  "24118478a12cd8e910ec3ae69edc8bda17c70754dd00d13f28dda0aa0f8644bb",
  "187bcbef9261e6c7eaefd8368e2b930a8bd7335cf541d8a05e9337beaf4c5f89",

  2,
  1,
  "0x6bECa9Eb5cE6b2Ad84910B952ea83109C277E30c",

# removeClaim

# getCredential 

dids.getCredential(identity, claimId);

Get claim detail of DID by claim id

# Parameters

  1. identity - string : DID contract address
  2. claimId - string : The claim id

# Returns

  1. claimType - number : The claim type
  2. scheme - Object : The claim scheme
  3. issuer - Object : The claim issuer
  4. signature - Object : A hex string of the signature
  5. data - Object : A hex string of the data

dids.getClaim("0x7Aaf39BBD7Dd755EaA50F992522A4Eb3d8cd75a0", "0x68ab8d5ae723515a5b77cb01cddf832ce9c6acfca1ab7bfaf8b912eab7b1667a");

# addKey 

dids.addKey(purpose, keyType, privateKeyAdd, privateKey);

Add key to Claim holder (only access for Management side)

# Parameters

  1. purpose - KeyPurposes : The key purpose (check Key purpose)
  2. keyType - Schemes : The key type (check Schemes)
  3. privateKeyAdd - string : The private key of account to add as key
  4. privateKey - string : Private key of account that has permission to interact with DID

# Returns

  1. receipt - object : A transaction receipt object, or null if no receipt was found. (check receipt)
  2. eventLogs - object : Event logs of transaction
  3. keyAdded - string : Added key of DID contract

dids.addKey(

# verifyCredential 

dids.verifyCredential(identity, claimType);

Check if DID has a claim of specific type (remember that our Claim Holder is static on server side)

# Parameters

  1. identity - string : DID contract address
  2. claimType - ClaimTypes : The claim type

# Returns

boolean : True if DID has a valid claim of specific type

# Example 

dids.checkClaim("0xece7519c282274542231d213A5919337c3F66686", 1);
> true
