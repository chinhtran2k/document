# DDR

This module use to mint/share/consent/lock DDR

# Main function

# mintDDR 

ddr.mintDDR(hashedData, ddrRawId, uri, patientDID, delegateKey, nonce);

Create DDR for Patients

# Parameters

  1. hashedData - string : Data was hashed by the keccak256
  2. ddrRawId - string : Id off-chain was sent from Pharumo
  3. uri - string (optional): The uri of DDR
  4. patientDID - string : Address of Patient was created from DID
  5. delegateKey - string : Private key of delegateKey
  6. nonce - number (optional): The nonce of account

# Returns

  1. receipt - object : A transaction receipt object, or null if no receipt was found. (check receipt).
  2. eventLogs - array : An array of event logs of transactions.
  3. ddrs - object : List of DDRs was minted

# Example

# getShareDDR 

ddr.getShareDDR(patientDID, ddrId);

Check if this address of patient is allowed to contact DDR

# Parameters

  1. patientDID - string : Address of Patient was created from DID
  2. ddrId - string : Id of DDR

# Returns

boolean : Check if address is allowed or not

# Example 

ddr.getShareDDR("0x7Aaf39BBD7Dd755EaA50F992522A4Eb3d8cd75a0", "A001").then(console.log);
> true;

# getConsentedDDR 

ddr.getConsentedDDR(providerDID, patientDID, ddrId);

Check if this address of provideris allowed to watch ddr

# Parameters

  1. providerDID - string : The DID address of provider
  2. patientDID - string : The DID address of patient
  3. ddrId - number : Id of ddr

# Returns

boolean : Check if address is allowed or not

# Example 

ddr.getConsentedDDR("0x37C819C94EdfE048968A28215C73C4D1B8ceAb15", "0x7Aaf39BBD7Dd755EaA50F992522A4Eb3d8cd75a0", "A001").then(console.log);
> true;

# getLockedDDR 

ddr.getLockedDDR(patientDID, ddrId);

Check if this DDR is locked or not

# Parameters

  1. patientDID - string : The DID address of patient
  2. ddrId - string : Id of ddr

# Returns

Boolean : True is locked and false is unlocked

# Example 

ddr.getLockedDDR("0x7Aaf39BBD7Dd755EaA50F992522A4Eb3d8cd75a0", "A001").then(console.log);
> true;
