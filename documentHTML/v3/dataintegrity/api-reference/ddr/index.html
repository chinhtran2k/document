# DDR

This module use to mint/share/consent/lock DDR.

# Main function

# mintDDR 

ddr.mintDDR(hashValue, ddrId, uri, patientDID, delegateKey, nonce?, isSimalate?);

Create DDR for Patients.

# Parameters

  1. hashValue - string : Data of DDR was hashed by the keccak256 from Pharumo.
  2. ddrId - string : DID Id off-chain was sent from Pharumo.
  3. uri - string (optional): The uri of DDR.
  4. patientDID - string : Address of Patient was created from DID.
  5. delegateKey - string : Private key of delegateKey.
  6. nonce - number (optional): The nonce of account.
  7. isSimulate - boolean (optional): Emulator status.

# Returns

  1. receipt - object : A transaction receipt object, or null if no receipt was found. (check receipt).
  2. eventLogs - array : An array of event logs of transactions.
  3. ddrs - object : List of DDRs was minted.

ddr.mintDDR(

# mintBatchDDR

# getDDR 

ddr.getDDR(patientDID, ddrId);

Get the ddr with certain ddr id.

# Parameters

  1. patientDID - string : Address of Patient was created from DID.
  2. ddrId - string : DDR Id off-chain was sent from Pharumo.

# Returns

  1. tokenId - string : Token Id of ddr.
  2. patientDID - string : Address of Patient was created from DID.
  3. ddrId - string : DDR Id off-chain was sent from Pharumo.
  4. hashValue - string : Data of DDR was hashed by the keccak256 from Pharumo.
  5. didConsentedOf - string : Array providers address shared by patient.

# Example 

ddr.getDDR("0x91a6eBbDF60316065a9b0ddBeBB10e0fb51F3218","001")
